I'm a Computer Science graduate (Zetech University) with a Data Science & Machine Learning certificate (Moringa School). I work across the full data pipeline — from cleaning messy raw data to building predictive models and shipping dashboards people actually use.

Recent work includes fraud detection, customer churn prediction, and transaction behavior forecasting, mostly with an African fintech/telecom lens.
